## What is TruthLens?

TruthLens helps you understand **why** a piece of content should be trusted, questioned, or verified — not just whether it passes a binary true/false check.

Online misinformation is rarely completely fabricated. More often, it is weakly sourced, emotionally manipulative, missing key context, or framed to mislead. TruthLens is built to slow that process down and surface the signals that matter.

## Who is it for?

TruthLens is built for anyone who needs signal clarity before sharing, citing, or publishing:

- **Students** — verify sources before using them in research
- **Journalists** — flag weak sourcing and emotional framing quickly
- **Researchers** — inspect claims for context gaps and unsupported assertions
- **Educators** — teach critical reading with real examples
- **Everyday readers** — slow down before resharing something viral

## What TruthLens analyzes

When you submit text, TruthLens evaluates:

- **Main claims** — what assertions is the content actually making?
- **Warning flags** — emotional language, weak sourcing, overconfident framing, missing context
- **Source quality** — are sources named, official, or verifiable?
- **Context gaps** — what baselines, timelines, or comparisons are absent?
- **Next steps** — concrete actions to verify or challenge the content
- **Live citations** — web-grounded sources retrieved at analysis time

## Tech stack

TruthLens is a Next.js application with a self-contained API layer:

| Layer | Technologies |
|---|---|
| Frontend | Next.js 16, React 19, TypeScript, Tailwind CSS |
| Primary AI | Google Gemini (`@google/generative-ai`) |
| Fallback AI | Azure AI Foundry — Llama 3.1 8B (`@azure-rest/ai-inference`) |
| Web grounding | Tavily (`@tavily/core`) |
| Database | Convex |
